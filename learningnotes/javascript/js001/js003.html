提取onclick进JS，必须调用函数。

JS里函数可以没有名字，需要把函数内容直接放到事件。例如：div1.onclick=function(){}

window.onload的作用是当页面加载完成后才执行这个函数的内容。

JS CSS HTM三者分离：不加行间JS事件，不加行间样式。提取行间事件就是为了三者分离。

getElementsByTagName，可以获取一组标签元素，所谓的TagName就是标签名，他的参数就是需要获取什么类型的元素，例如div。他所获取到的就不是一个元素了，而是一组。可以通过length属性统计个数。getElementById，每次只能获取一个ID元素。
